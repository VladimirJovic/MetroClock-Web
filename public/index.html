Integrations

Connect MetroClock with your team's tools

💬

Slack

Send notifications to a Slack channel

How to set up: In Slack, go to Apps → Incoming Webhooks → Add to Slack, choose a channel, and paste the webhook URL above.
User Mapping — Slack

Enter each employee's Slack Member ID so they get tagged in notifications. Find it in Slack: click on a user → View full profile → More → Copy member ID.

🎮

Discord

Send notifications to a Discord channel

How to set up: In Discord, go to Server Settings → Integrations → Webhooks → New Webhook, choose a channel, and paste the URL above.
User Mapping — Discord

Enter each employee's Discord User ID so they get tagged in notifications. Enable Developer Mode in Discord (Settings → Advanced), then right-click a user → Copy User ID.
